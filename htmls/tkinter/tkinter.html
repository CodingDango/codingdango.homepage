Periodic Table GUI

A GUI program using the tkinter library.
A simple test app that makes you recite the element symbols, and element names.

Features

Has a lives function, maximum of 3 mistakes before game over,
lives are reset after each correct answer.
Also includes a hint function whenever you get the current element wrong,
revealing the first letter of the element name,
and as more mistakes pile up, more of the letters following the revealed letter are shown.

The periodic table is in CSV format, to allow for parsing and checking for the correct answer.
